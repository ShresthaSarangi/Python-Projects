Caesar Cipher (Python)

A simple Python program that encrypts and decrypts messages using the Caesar Cipher technique.

Features

- Encrypt messages
- Decrypt messages
- Works with uppercase and lowercase letters
- Handles non-alphabet characters

How It Works

The Caesar Cipher shifts each letter in a message by a specific number of positions in the alphabet.

Example:
Message: HELLO
Key: 3
Encrypted: KHOOR

How to Run

1. Clone the repository
2. Navigate to the folder
3. Run the program

python caesar_cipher.py
