# 🔐 Secure Offline Password Vault

A simple command-line password manager built in Python to securely store, view, and manage credentials locally.

---

## 🚀 Features

- ➕ Add new credentials (website, username, password)
- 👀 View saved credentials
- 🔄 Update existing passwords
- 💪 Password strength checker
- 🔐 Encoded storage using Base64 (for learning purposes)

---

## 🛠️ Tech Stack

- Python 3
- File Handling
- Base64 Encoding

## ⚠️ Disclaimer

This project uses Base64 encoding, which is **not secure encryption**.  
It is built for learning purposes only and should not be used to store real passwords.
