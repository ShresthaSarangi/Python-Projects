# 🌦️ Real-Time Weather API Logger

A simple Python application that fetches real-time weather data using an API and logs it into a JSON file for easy storage and access.

---

## ⚙️ Functions

- Fetch real-time weather data from an API  
- Extract key details like temperature, humidity, and weather condition  
- Store data in a structured JSON format  
- Append new weather logs without overwriting previous data  
- Allow easy reading and tracking of stored weather data
