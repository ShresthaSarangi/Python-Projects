📊 CSV to JSON Converter (Python)

A simple command-line Python tool that converts CSV files into JSON format.
This project is designed for beginners to understand file handling, data processing, and working with structured data.

---

✨ Features

- 📥 Load data from a CSV file
- 🔄 Convert CSV data into JSON format
- 👀 Preview records before conversion
- 📄 Pretty formatted JSON output
- 🖥️ Simple and interactive CLI interface

---

🛠️ Technologies Used

- Python 3
- Built-in modules:
  - "csv"
  - "json"
  - "os"

---

📂 Project Structure

CSV-to-JSON-Converter/
│
├── csv_to_JSON.py        # Main script
├── raw_data.csv         # Sample input file
├── converted_data.json  # Output file
└── README.md


---

🧪 Sample CSV Format

Make sure your CSV file looks like this:

name,age,city
John,25,New York
Alice,30,London

---

⚠️ Notes

- Ensure the CSV file exists before loading data
- JSON file will be created automatically after conversion
- Use UTF-8 encoding for proper character handling
